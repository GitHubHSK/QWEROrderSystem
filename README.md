# QR Order System

## プロジェクト概要

QRコードを読み取ることで、お客様がスマートフォンからメニューを閲覧し、注文できるWebサービスです。

## 使用技術

### Backend
- Java
- Spring Boot
- JPA

### Frontend
- JavaScript
- Thymeleaf

### Database
- MySQL

## 主要機能

### 利用者向け機能
- QRコードからメニューへアクセス
- テーブル選択
- 利用人数の選択
- アレルギー情報の入力
- カテゴリ別メニュー閲覧
- 商品注文
- 注文内容の一時保存
- 注文金額の自動計算

### 管理者向け機能
- 管理者ログイン
- メニュー登録・編集・削除
- カテゴリ管理
- 注文状況確認

## 担当役割
- チームリーダー
- 進捗管理
- DB設計
- 設計書作成
- ログイン機能実装
- メニュー管理機能実装

## 工夫した点
- Ajaxを利用し、ページ遷移なしでメニュー情報を更新できるようにした。
- セッションを利用して注文情報を保持し、ユーザーの利便性を向上させた。
- チームリーダーとして進捗管理とタスク分担を行った。
